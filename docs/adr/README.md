## Note on Numbering Gaps

ADR numbers are not contiguous (e.g., 007, 008, 015). This is intentional: early architectural decisions (ADR-001 through ADR-006, ADR-009 through ADR-014) were made verbally or in chat during the initial project setup phase and were never formally recorded. Only decisions made after the ADR process was established are documented here.
