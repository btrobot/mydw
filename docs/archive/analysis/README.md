# Archive Analysis Index / 历史分析索引

`docs/archive/analysis/` 保存的是**早期分析稿、ER/操作分析、历史设计判断**。

当前主要内容：

- `domain-model-analysis.md` — 早期领域模型迁移分析稿
- `refactor-list.md` — 早期 refactor 清单
- `task-management-analysis.md` — task 管理分析稿
- `task-management-er-design.md` — task 管理 ER 设计稿
- `task-management-operations.md` — task 管理操作分析

适合看这些文档的场景：

- 需要理解当时“为什么会这么设计”
- 想回溯某轮分析判断的上下文
- 想比对旧分析与当前 truth 的差异

不适合把这里当作：

- 当前领域模型说明
- 当前 task 语义说明
- 当前架构边界说明

当前真相优先看：

- `docs/current/architecture.md`
- `docs/current/runtime-truth.md`
- `docs/domains/`
