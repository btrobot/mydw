# Archive Reference Index / 历史参考索引

`docs/archive/reference/` 保存的是**旧事实参考长文**，不是当前 authoritative source。

当前主要内容：

- `system-architecture.md` — 旧的全面架构说明
- `api-reference.md` — 旧 API 参考
- `data-model.md` — 旧数据模型字典
- `coze-integration.md` — 偏离当前主线的历史接入说明

适合在这些场景下阅读：

- 需要理解早期系统是怎么被解释的
- 需要对照老字段 / 老接口 / 老架构表述
- 需要追溯某个历史 integration 方案

不适合把这里当作：

- 当前默认架构入口
- 当前 API truth
- 当前数据库 truth

当前真相优先看：

- `docs/current/architecture.md`
- `docs/current/runtime-truth.md`
- `/docs`
- `/openapi.json`
- `backend/models/__init__.py`
