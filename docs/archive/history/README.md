# Archive History Index / 历史收口索引

`docs/archive/history/` 保存的是**阶段收口、roadmap、gap tracking、历史性证明文档**。

当前主要内容：

- `frontend-ui-ux-closeout-final-summary.md` — Frontend UI/UX closeout 历史收口总结
- `refactor-roadmap.md` — 历史 refactor roadmap
- `refactor-issue-breakdown.md` — 历史问题拆解
- `refactor-gap-list.md` — 历史 gap list

适合看这些文档的场景：

- 回顾某一阶段当时的完成度判断
- 做 closeout / audit / retrospective
- 对照历史 roadmap 与当前 backlog

不适合把这里当作：

- 当前 backlog
- 当前阶段启动入口

当前主线与下一阶段入口优先看：

- `docs/governance/next-phase-backlog.md`
- `docs/current/next-phase-kickoff.md`
