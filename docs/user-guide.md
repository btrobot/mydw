## 1. 安装与启动

### 环境要求

- 操作系统：Windows 10 及以上
- 浏览器：工具内置 Chromium，无需额外安装
- 网络：需能正常访问得物创作者平台（creator.dewu.com）

### 启动步骤

1. 双击启动得物掘金工具
2. 等待系统初始化完成，进入主界面

---

## 2. 账号管理

### 2.1 添加账号

1. 点击左侧导航栏「账号管理」
2. 点击「添加账号」按钮
3. 填写账号 ID 和账号名称
4. 可选：添加标签和备注，方便后续管理
5. 点击确认完成添加

### 2.2 登录得物账号

1. 在账号列表中找到目标账号，点击「连接」按钮
2. 在弹出的连接窗口中输入手机号
3. 点击「发送验证码」，输入收到的短信验证码
4. 等待验证完成，账号状态变为绿色「已连接」

### 2.3 检查连接状态

- 已连接的账号显示「检查连接」按钮，点击可验证 Session 是否有效
- Session 过期后按钮变为「重新登录」，点击重新走登录流程
- 点击工具栏「批量检查连接」可一次性检查所有账号状态

### 2.4 预览账号

1. 对已连接的账号，点击「预览」按钮
2. 系统打开可见浏览器窗口，展示该账号在得物创作者平台的实际页面
3. 可用于确认登录状态、查看账号信息
4. 使用完毕后点击「关闭预览」，可选择是否保存当前 Session

### 2.5 删除账号

1. 点击账号操作列的「删除」按钮
2. 确认删除后，账号及其关联数据将被移除

## 3. 素材准备

### 3.1 寻找爆款素材

1. 在得物 App 或网页端浏览热门视频
2. 找到与目标商品相关的爆款视频
3. 记录商品链接、视频标题、视频封面

### 3.2 保存素材到本地

1. 将视频文件下载到本地素材目录
2. 在「素材管理」页面点击「扫描素材」自动导入
3. 也可手动点击「导入素材」选择文件

### 3.3 添加商品

1. 在「素材管理」页面的商品区域点击「添加商品」
2. 填写商品名称和链接
3. 商品信息将在发布时自动关联到视频

## 4. AI 剪辑合成

### 4.1 创建剪辑任务

1. 进入「AI 剪辑」页面
2. 选择 2 个或以上的源视频进行合成
3. 设置合成参数：

| 参数 | 说明 | 建议值 |
|------|------|--------|
| 合成时长 | 最终视频时长 | 2-5 分钟 |
| 素材时长要求 | 单个素材最短时长 | 30 秒以上 |
| 文案模式 | 勾选「原文案」使用源视频文案；勾选「自动文案」由系统生成 | 按需选择 |

### 4.2 开始合成

1. 参数设置完成后，点击「添加任务」
2. 点击「开始合成」
3. 合成过程大约 1-2 分钟
4. 合成完成后，视频出现在素材列表中

### 4.3 提取视频信息

- 合成完成后点击「提取」按钮
- 系统自动提取视频标题和文案，用于后续发布

## 5. 话题设置

### 5.1 搜索热门话题

1. 在话题输入框中输入关键词
2. 系统显示相关话题及其热度
3. 优先选择热度较高的话题

### 5.2 设置发布话题

1. 从搜索结果中选择约 5 个相关话题
2. 选定后，后续发布的所有视频都会自动带上这些话题
3. 如需更换，重新搜索并选择即可

## 6. 任务管理与发布

### 6.1 创建发布任务

1. 进入「任务管理」页面
2. 选择要发布的视频素材
3. 关联对应的商品和账号
4. 系统自动填入标题、文案和话题

### 6.2 定时发布设置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 发布日期 | 开始发布的日期 | 今天 |
| 每日发布数量 | 每天发布几条 | 6 条 |
| 发布时间段 | 允许发布的时间范围 | 9:00 - 22:00 |
| 发布间隔 | 相邻两条之间的等待时间 | 30 分钟 |

设置完成后点击「保存配置」。

### 6.3 开始发布

- **顺序发布**：按设定的时间间隔依次发布
- **直接发布**：立即发布所有待发布任务

发布时系统自动完成以下操作：
1. 打开对应账号的浏览器
2. 上传视频文件
3. 填入标题、文案、话题
4. 关联商品链接
5. 完成发布

### 6.4 查看发布状态

任务列表实时显示每条任务的状态：

| 状态 | 说明 |
|------|------|
| 待发布 | 等待发布时间到达 |
| 发布中 | 正在上传和发布 |
| 已成功 | 发布完成 |
| 已失败 | 发布出错，可查看原因并重试 |

## 7. 开通视频号佣金

### 7.1 达标要求

得物视频号佣金开通需满足以下指标：

| 指标 | 说明 |
|------|------|
| 播放量 | 视频累计播放量达标 |
| 收藏量 | 视频累计收藏数达标 |
| 点击量 | 商品链接累计点击数达标 |

### 7.2 开通步骤

1. 账号指标达标后，进入得物创作者平台
2. 找到「开通视频号」入口
3. 确认无违规作品（前提条件）
4. 点击「我要加入」完成开通

### 7.3 注意事项

- 账号必须无违规记录
- 建议先积累一定数量的优质作品再申请
- 开通后即可通过视频带货获得佣金收入

## 常见问题

**Q: 登录后显示 Session 已过期怎么办？**

点击「重新登录」重新输入验证码。建议定期使用「批量检查连接」检查所有账号状态。

**Q: 视频合成失败怎么办？**

检查源视频文件是否完整、时长是否满足 30 秒以上。问题持续则尝试更换素材重新合成。

**Q: 发布失败怎么办？**

1. 检查账号连接状态是否正常
2. 检查视频文件是否存在
3. 查看任务详情中的错误信息
4. 重试发布或重新登录后再试

**Q: 可以同时管理多少个账号？**

支持多账号管理，但建议控制在合理范围内。批量操作时系统串行执行并自动控制间隔，避免触发平台风控。
